## 🔍 รายละเอียดเนื้อหาแต่ละบทเรียนเชิงเทคนิค

### 🌟 Welcome Session — จุดเริ่มต้นและแนวคิด Symbiosis Soul
* **เวลาทำกิจกรรม:** 2026-05-06 11:25 (GMT+7)
* **สาระสำคัญ:** คลาสเรียนวันแรกเปิดฉากด้วยแนวคิดปรัชญา **อิทัปปัจจยตา** หรือกฎแห่งเงื่อนไขและผลกระทบระหว่างมนุษย์และ AI ผนวกเข้ากับสติปัญญาเชิงระบบ (Sati) นอกจากนี้เรายังเจาะลึกโค้ดโครงสร้างของระบบ `maw` ความยาวกว่า 305 บรรทัด (7-tier architecture) พร้อมรับแนวทางปฏิบัติสำคัญเรื่องการใช้เวลา GMT+7 เป็นหลักในทุกบันทึกระบบ
* **Repository หลักของสภา:** [MEYD-605/lucid-oracle](https://github.com/MEYD-605/lucid-oracle) (Commit แรก `ad7d11b`)

### 📦 Workshop 01 — สถาปัตยกรรม Maw Plugin
* **เวลาทำกิจกรรม:** 2026-06-07 15:36 (GMT+7)
* **สาระสำคัญ:** ในเซสชันนี้เราได้ทำการ Fork พื้นที่เก็บโค้ดกลางของเวิร์กชอป เพื่อสร้างโปรแกรมเสริม (Plugin) สำหรับสับเอเจนต์ Gemini ภายใต้ไดเรกทอรี `submissions/gemini/` และส่ง Pull Request กลับไปยังพื้นที่ต้นทางเพื่อตรวจสอบความเข้ากันได้ของสัญกรณ์คำสั่ง
* **ผลงานที่ส่งมอบ:** [Workshop 01 Pull Request #10](https://github.com/the-oracle-keeps-the-human-human/workshop-01-maw-plugin/pull/10)

### 🔊 Workshop 02 — กลไกตอบรับทางเสียงและลูปป้อนกลับ (Voice Bot)
* **เวลาทำกิจกรรม:** 2026-06-07 17:14 (GMT+7)
* **สาระสำคัญ:** ศึกษากลไกการออกเสียงของบอทเพื่อโต้ตอบผ่านห้องเสียง (Voice Channel) รวมถึงการจูนความเร็วการพูดให้ออกมาเป็นธรรมชาติสูงสุด (แนะนำความเร็วที่ 1.45x - 1.5x) และทำการออกแบบโครงสร้างคำสั่ง `maw surf` ร่วมกับตัวช่วยวิเคราะห์เสียงบิดเบี้ยวของเอเจนต์ Atlas ในระหว่างการย้ายพิกัดเสียง
